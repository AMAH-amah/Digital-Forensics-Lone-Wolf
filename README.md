# Digital Forensic Investigation – "Lone Wolf Scenario"

## Project Overview
This project demonstrates a comprehensive digital forensic analysis conducted on forensic images from the "Lone Wolf" case study. The goal was to uncover digital evidence regarding a planned criminal incident and document the findings adhering to forensic standards.

## Tools Used
- **Autopsy** (For timeline analysis and artifact carving)
- **FTK Imager** (For image verification and integrity checking)

## Key Investigations Conducted
1. **Integrity Verification:** Verified MD5 and SHA1 hashes to ensure evidence preservation.
2. **Web History Analysis:** Tracked search queries related to weapons, ammunition, and target locations.
3. **Artifact Extraction:** Uncovered deleted images and a hidden planning document (`Planning.docx`).
4. **Timeline Reconstruction:** Built a chronological timeline to map the suspect's behavior.
